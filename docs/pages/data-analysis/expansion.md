# 扩展

## 技巧补充

1. 一个数 × 1.5 -> 自身 + 自身的一半
2. 一个数 × 1.1 -> 错位相加
3. 一个数 × 0.9 -> 错位相减

1/2 = 50%

1/3 ≈ 33.3%

1/4 = 25%

1/5 = 20%

1/6 ≈ 16.7%

1/7 ≈ 14.3%

1/8 = 12.5%

1/9 ≈ 11.1%

1/10 = 10%

1/11 ≈ 9.1%

1/12 ≈ 8.3%

1/13 ≈ 7.7%

1/14 ≈ 7.1%

1/15 ≈ 6.7%

1/16 = 6.3%

1/17 ≈ 5.9%

1/18 ≈ 5.6%

## 百化分的公式

$$
n=\frac{100}{百分号前的数字}（保留小数点后一位）
$$
